# Simple Interest Calculator

A simple and interactive web-based calculator for computing simple interest.

## Features

- **Calculate Simple Interest**: Quickly compute simple interest using Principal, Rate, and Time
- **User-Friendly Interface**: Clean and intuitive design for easy calculation
- **Real-time Results**: Displays Principal, Simple Interest, and Total Amount
- **Reset Functionality**: Clear all fields with a single click
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Enter Key Support**: Press Enter to calculate results

## Formula

The simple interest is calculated using the formula:

```
Simple Interest (SI) = (P × R × T) / 100

Where:
- P = Principal Amount (initial sum of money)
- R = Rate of Interest per annum (%)
- T = Time Period (in years)

Total Amount = Principal + Simple Interest
```

## How to Use

1. Enter the **Principal Amount** (in dollars)
2. Enter the **Rate of Interest** (% per annum)
3. Enter the **Time Period** (in years)
4. Click the **Calculate** button or press **Enter**
5. View the results displayed below:
   - Principal Amount
   - Simple Interest earned
   - Total Amount (Principal + Interest)
